## 💡 Points Clés à Retenir

### Architecture
```
appState (state.js)
    ↓
3 calculateurs clé:
├─ tariffEngine.js (costs)
├─ pvSimulation.js (solar)
└─ tempoCalendar.js (tempo colors)
    ↓
10 display functions (app.js)
    ↓
UI (index.html + style.css)
```

### Workflow
```
User imports file
  ↓
triggerFullRecalculation():
  ├─ Parse JSON/CSV
  ├─ Compute 5 tariffs
  ├─ Display 7 charts
  ├─ If PV: compute savings
  ├─ Display 3 more charts
  └─ Save to localStorage
```

### localStorage
```
'theme' → 'light' | 'dark'
'comparatifElec.pvSettings' → {6 values}
'comparatifElec.tempoDayMap' → {day colors}
```
