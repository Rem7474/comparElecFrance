1) Télécharger les 52 fichiers hebdomadaires

Entrez le PRM (identifiant de votre compteur). Le bouton ci‑dessous génère un script JavaScript à coller dans la console du site Enedis (F12 → Console). Ce script récupérera automatiquement l'idPersonne via l'endpoint utilisateur et téléchargera/assemblera les 52 exports hebdomadaires.

Après avoir généré le fichier via le script console (consommation_annee.json), sélectionnez-le ici. L'analyse démarre automatiquement à l'import.

Chargez ci‑dessous le fichier annuel (généré par le script console ou par la fusion). L'outil calcule la consommation annuelle et affiche la distribution horaire moyenne, ainsi que min/max par heure.
